# Isomorphic
Minimalist browser for web-based desktop apps, based on WebView

# Installation

```
git clone https://github.com/serge-hulne/isomorphic
cd isomorphic
make
make install
```

(on Linux : `sudo make install` (Ubuntu) or `make install` as root (Debian))
